# ChainVote Backend

Scalable backend for the ChainVote smart contract using ethers v6.

## Structure

- `config/`: Provider and wallet configuration
- `contracts/`: Ethers contract instance and ABI
- `services/`: Core logic (vote, read, events)
- `scripts/`: Executable runners
- `utils/`: Shared utilities (logger)

## Run

```bash
npm install
npm run vote
npm run read
npm run events
```
